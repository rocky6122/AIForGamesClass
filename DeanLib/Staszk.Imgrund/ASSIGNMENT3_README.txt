Commands to control the flocking Boids:
Caution: Dean's code crashes after 100 Boids. So watch out.

A: Adds 10 flocking boids

D: Deletes the most recent Boid

ESC: Quits the program

S: Saves the changable values to a file

L: Loads the changable values from a file if there is one

Keypad 7: Subtracts Cohesion Weight

Keypad 9: Adds Cohesion Weight

Keypad 4: Subtracts Separation Weight

Kaypad 6: Adds Separation Weight

Keypad 1: Subtracts Velocity Matching Weight

Keypad 3: Adds Velocity Matching Weight
